
# Debian Setup script

Used to quickly setup all the normal stuff that you'd expect to see on a server. 

 - Default JDK
 - Default JRE
 - Python3
 - Python3 Pip
 - fail2ban
 - ufw
 - git
 - nodejs
 - npm
 - libssl-dev
 - software common properties
 - build-essential (GCC)
 - manpages-dev
 - podman
 - podman-compose
 - docker
 - docker-compose


## Using this Script

pretty on the nose but regardless:

`git clone https://github.com/gnubyte/debian-setup.git`

then 

`chmod +x setup.sh`

then

`./setup.sh`
